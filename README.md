# 🚗 Automobile Sales Performance Dashboard

## Project Overview
This project is an interactive Power BI dashboard designed to analyze automobile sales performance and provide business insights through data visualization.

## Dashboard Features
- 🎛 Interactive Slicers
- 💰Total Revenue KPI
- 🚗 Cars Sold KPI
- 💵 Average Price KPI
- 🏢 Number of Companies KPI
- 📈 Revenue by Company
- 🥧 Transmission Distribution
- 🚘 Revenue by Model 
- 👥 Revenue by Gender
- 🌍 Revenue by Dealer Region
- 🎯 Revenue Achievement Gauge

## Tools Used
- Power BI
- DAX
- Data Visualization
- Data Analysis

## Key Insights
- Identified top revenue-generating companies.
- Analyzed revenue distribution across dealer regions.
- Compared sales performance by gender.
- Evaluated transmission preferences.
- Tracked revenue achievement against target.
